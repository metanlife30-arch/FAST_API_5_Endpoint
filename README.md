Стек технологий:
    Python 3.11
    FastAPI
    SQLAlchemy
    Alembic
    PostgreSQL 16
    Docker + Docker Compose
    Pydantic
    Uvicorn ASGI Server

1. Собрать и запустить проект командой
    docker-compose up --build

2. Автоматически применятся все миграции

3. Зайти по ссылки http://127.0.0.1:8000/docs
